Welcome To The Number Match Game!

The rules are simple.

  1. A random image and number will be generated.
  2. If the number on the screen matches the number of items in the image, hit the thumbs up button.
  3. If the number on the screen does not match the number of items in the image, hit the thumbs down button.
  4. You'll have seconds to decide.
